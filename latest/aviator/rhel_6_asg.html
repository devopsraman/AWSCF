Rhel6Asg FAWS Template

rhel_6_asg.template - Autoscaling EC2 Instances with Rolling Updates and Notifications. WARNING This template creates one or more Amazon EC2 instances and an Elastic Load Balancer. You will be billed for the AWS resources used if you create a stack from this template. ### Parameters

  • Default: 20M
  • Type: String
  • ConstraintDescription: #H#M#S where each # is the number of hours or minutes or seconds
  • Description: Time to wait for the number of signals equal to ScalingMin. H/M/S Hours/Minutes/Seconds

  • Default: ``
